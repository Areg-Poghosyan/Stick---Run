Stick & Run

A stick figure platformer with three worlds, a cinematic story, and an ending you won't see coming.

Created by Areg Poghosyan


Play

Open index.html in any modern browser. No installation or internet connection required.

Available on desktop and mobile.


Overview

Stick & Run is a hand-crafted 2D platformer built entirely in HTML5 Canvas with no external libraries. You play as a stick figure navigating three increasingly dangerous worlds, each with its own visual identity, music, and atmosphere.

The game has a story. Pay attention to the NPCs.

How to Play


Collect all coins before reaching the finish line — the trophy won't activate otherwise
You have 3 lives and a countdown timer per level
Activate the checkpoint flag mid-level to set your respawn point
Springs launch you higher than a normal jump
Wind zones push you horizontally or vertically — learn to use them
Enemies knock off a life on contact



The Three Worlds

Level 1

A bright open world with jazz music and a friendly NPC who explains the rules. A good starting point.

Level 2

A darker, more dangerous world. The platforms are tighter, the enemies are faster, and the music shifts. Something feels different.

Level 3

The hardest world. The atmosphere changes completely. Two NPCs say things that don't quite make sense — until they do.

Reach the end and find out what the game is really about.


Features


3 full levels with unique visuals, music, enemies, and layouts
Cinematic story with NPC dialogue and a narrative ending
Character customization — choose your stick figure's colors before playing
Main menu, pause menu, level select screen
Checkpoints, springs, wind zones, moving platforms
Dynamically generated music per level using the Web Audio API
Mobile support with on-screen touch controls
Capped at 60fps regardless of monitor refresh rate



Technical


Pure HTML5 / JavaScript / Canvas — no frameworks or libraries
All audio generated in real time via the Web Audio API
Single file — everything lives in index.html
Runs locally with no server or internet connection required
